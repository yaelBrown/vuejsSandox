  Known options for Vue Instance
    el - Connects to DOM
    data - Store Data to be used
    methods - Methods of this Vue Instance
    computed - Dependent properties
      can used like a property in the data object
      working with dependencies
    watch - excute code upon data changes

  Directives - special token in the markup that tells the library to do something to a DOM element
    v-on:input - has to be es5
    v-once: - Stick to initial value (read only)
    v-bind:href - binds data dynamically
    v-html - render html code and not escape it
      Does expose you to XSS

    v-model - used for 2 way binding. Updates the variable and where that variable is used throughout the application

  {{}} - syntax is called Interpolation or String Interpolation

  shorthand
    v-on = @
      v-on:click = @click
    v-bind = :
      v-bind:href = :href

  CSS is same as with vanilla
    scoped - encapsulates style to one particular component.
      <style scoped></style>

  Vue-Cli 3
    vue's command line interface
    tool for generating and building vue apps
    includes babel, typescript, eslint, postCSS and more
    includes dev server with hot reload
    includes Vue UI tool to manage app with GUI

    npm install -g @vue/cli
      installs latest version of vue cli globally

    vue create [name]
      creates app with vue cli cmd

    vue ui
      creates application with ui interface

    npm run serve
      starts the development server

    main.js - entry point for vue app
    App.vue - is a component

  Vuex for state management
    manage application level state.
      similar to redux or flux
    central store for all components
    uses actions to update global state and send down to components
    used in larger applications
